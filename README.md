### Weather Data API
The Weather API gathers station and date from Europe and returns a response with the temperature

Endpoints:   
URL format: http://127.0.0.1:5000/api/v1/station/date   
URL example: http://127.0.0.1:5000/api/v1/10/1988-10-25   
URL example for one station for one date: http://127.0.0.1:5000/api/v1/10/1988-10-25   
URL example for one station for all dates: http://127.0.0.1:5000/api/v1/10   
URL example for one station for one specific year: http://127.0.0.1:5000/api/v1/yearly/10/1985
